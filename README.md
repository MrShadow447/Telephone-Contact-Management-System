# 📞 Telephone Contact Management System (C)

## 📌 Overview

This project is a simple Telephone Contact Management System developed using the C programming language. It allows users to store and manage contact details efficiently using basic data structures.

The system provides functionality to add, delete, search, update, and display contacts in an organized way.

---

## 🚀 Features

* Add new contacts
* Delete existing contacts
* Search for a contact
* Update contact details
* Display all contacts

---

## 🧠 Data Structures Used

* **Array** – To store multiple contacts
* **Structure** – To represent each contact (name and phone number)
* **Linear Search** – To find contacts

---

## ⚙️ How It Works

The program is menu-driven. Users select an option from the menu, and the system performs the corresponding operation.

## ▶️ How to Run

1. Copy the code into a `.c` file (e.g., `contact.c`)
2. Compile the program:

   ```
   gcc contact.c -o contact
   ```
3. Run the executable:

   ```
   ./contact
   ```
